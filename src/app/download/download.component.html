Downloads

The Complex Portal is released monthly and data on individual complexes is available for download in both PSI-MI XML2.5 and XML3.0 formats (see the PSI website for a detailed format description). The files are grouped by species. We have now also developed ComplexTAB, a flatfile format to enable simpler parsing or loading into Microsoft Excel. A separate ComplexTAB file has been produced for each species - details on the format are available here . All files are available from the ftp site and will be updated as we add new complexes with every data release.
